## 📋 核心知识点

### 基础命令
```bash
git init                    # 初始化仓库
git add <file>              # 添加到暂存区
git commit -m "描述"        # 提交到仓库
git status                  # 查看状态
git log                     # 查看历史
```

### 分支管理
```bash
git branch                  # 查看分支
git checkout -b <name>      # 创建并切换分支
git merge <branch>          # 合并分支
```

### 版本回滚
```bash
git reset --hard <commit>   # 回滚到指定版本
git reflog                  # 查看所有操作历史
```
